Integration guide

From install to production in four steps.

The public API stays intentionally small. Import the standalone component, describe a typed menu tree, bind controlled state and adapt the design with CSS custom properties.

  1. 1

    Install

    npm install @ramiz4/ngx-multi-level-push-menu

  2. 2

    Import

    Add MultiLevelPushMenuComponent to the component imports.

  3. 3

    Bind

    Pass menu, options and optionally two-way bind collapsed.

  4. 4

    Observe

    Use typed activation outputs for analytics, authorization and application state.

<ngx-multi-level-push-menu [menu]="menuItems" [options]="options" [(collapsed)]="collapsed" (itemActivate)="onActivate($event)" />
